# Boucles - TP6 - Série harmonique

## Consignes

Créer une fonction `harmonique(n: number): number` calculant la somme des n premiers termes de la "série harmonique".

## Rappels mathématiques

La série harmonique d'un nombre naturel non nul n est la série des inverses des entiers naturels non nuls :

$\sum^{\infty}_{n=1} \frac{1}{n} = 1 + \frac{1}{2} + \frac{1}{3} + \dots + \frac{1}{n}$.

## Exemples

- `harmonique(1)` $= 1$
- `harmonique(2)` $= 1.5$
- `harmonique(3)` $\approx 1.8$
- `harmonique(4)` $\approx 2.1$
